INSTALAÇÃO

O NOD32® Sistema Antivírus é atualmente composto por diversos módulos
de programa, que são controlados através do NOD32 Control Center.
Nós selecionamos padrões de configuração que são apropriados para todos
os usuários, até mesmo os usuários mais avançados. Nós recomendamos
que durante a instalação as configurações padrão sejam selecionadas.

Durante a instalação, as características padrão serão instaladas,
adequando-se a configuração de seu computador. 

Instalações em rede possuem componentes adicionais, é aconselhável uma
revisão do "Guia de Instalação para Administradores de Rede" pelo
administrador do sistema, a fim de reforçar as vantagens oferecidas
por essas características adicionais.
